# BookStore FrontEnd

### Set Up Instructrion

1. Clone The Project
```
    git clone https://github.com/manish25Coder/BookStore-Frontend.git
``` 

2. Move Into The Directory 
```
    cd BookStore_Frontend
```

3. Install The Dependencies 
```
    npm i
```

4. Run The Server 
```
    npm run dev
```
